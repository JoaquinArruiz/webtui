## Reference

### Properties

Use CSS custom properties to customize the range appearance

- `--range-track-color`: Color of the slider track
- `--range-thumb-color`: Color of the slider thumb
- `--range-progress-color`: Color of the progress/filled portion
- `--range-track-height`: Height of the track
- `--range-thumb-size`: Size of the thumb

```css
#my-range {
  --range-track-color: var(--background2);
  --range-thumb-color: var(--foreground0);
}
```

### Attributes

- `size-="small"` - Smaller range slider (10ch width)
- `size-="large"` - Larger range slider (30ch width)
- `thumb-="small"` - Smaller thumb (0.6ch size)
- `thumb-="medium"` - Medium thumb (1ch size, default)
- `thumb-="large"` - Larger thumb (1.4ch size)
- `bar-="thick"` - Thicker track (1lh height)
- `bar-="line"` - Thin line track (1px height)


### Extending

To extend the Range stylesheet, define a CSS rule on the `components` layer

```css
@layer components {
    input[type="range"] {
        &[size-="xl"] {
            width: 40ch;
            --range-thumb-size: 1.5ch;
        }

        &[variant-="accent"] {
            --range-track-color: var(--accent-color);
            --range-thumb-color: var(--accent-foreground);
        }

        /* ... */
    }
}
```

### Scope

- All `<input type="range">` elements

```css
input[type="range"] { /* ... */ }
```
